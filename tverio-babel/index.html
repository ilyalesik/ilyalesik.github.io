Что такое Babel?

JavaScript компилятор

  1. Трансформирует ES 2015+, JSX, Flow, TS → ES5
  2. Модифицирует код (codemod)
  3. Полифилит фичи
  4. Можно расширять плагинами

Как Babel трансформирует код?

  1. Parse
    1. Lexical Analysis: source code → stream of tokens
    2. Syntactic Analysis: stream of tokens → AST
  2. Traverse AST: AST → AST*
  3. Generate: AST* → generated code

AST - Abstract Syntax Tree
