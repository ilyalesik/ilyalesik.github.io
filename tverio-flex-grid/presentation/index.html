Как заверстать сложный лейаут

  • Постарайтесь не использовать вложенных контейнеров
    • .container__left-side, .container__right-side — плохо
  • Начните с флексов
  • Если больше одной строки, и строки имеют разную конфигурацию — возьмите гриды

Grid: два подхода

Без grid-area

Больше подходит для раскладок с неизвестным количеством элементов (grid-auto-flow, grid-auto-rows, grid-auto-columns)

С использованием grid-area

Более декларативно: для изменения конфигурации меняем свойства контейнера, а не элементов
